# Learning Python, 6th Edition

Study workspace for *Learning Python, 6th Edition* by Mark Lutz.

Source reference: O'Reilly lists the 6th edition as a 2025 update focused on the core Python language, efficient code, syntax, objects, modules, classes, and related tools.

## What To Put In Each Part

- `chapter_xx.md`: your explanation of the chapter.
- `examples/`: code copied from memory after reading.
- `exercises/`: small drills for that chapter.
- `summary.md`: what you can explain without notes.

## Learning Standard

A chapter is complete only when you can:

1. Explain the idea simply.
2. Write a runnable example without looking.
3. Name the common mistakes.
4. Use the concept in a small project.
