# Glossary

Write terms in your own words. Keep definitions short and concrete.

| Term | Meaning | Example |
| --- | --- | --- |
| Object | A value Python can store, pass, inspect, or operate on. | `42`, `"name"`, `[1, 2]` |
| Reference | A name or container slot pointing to an object. | `x = [1, 2]` |
| Mutable | Can be changed in place. | `list`, `dict`, `set` |
| Immutable | Cannot be changed in place. | `int`, `str`, `tuple` |
