# 00 — Foundations

**Goal of this module:** by the end, you should be able to answer, *out loud*, these four questions:

1. What does a computer actually do when it "runs" code?
2. Why was Python invented, and what makes it different from C or Java?
3. What happens between you typing `python script.py` and seeing output?
4. When a Python program is running, what is "in memory"? What is `x = 5` actually doing?

If you can't answer #4 in particular, the rest of the curriculum becomes a memorization exercise. Spend the time here.
