# Learning Python 6e Projects

Use this folder for applied mini-projects.

Project ladder:

1. CLI notes app - variables, strings, lists, files.
2. Expense tracker - dictionaries, functions, files, JSON.
3. Quiz runner - modules, packages, control flow.
4. Contact book - classes, exceptions, persistence.
5. Log analyzer - iteration, generators, text processing.

Each project should include:

- `README.md`
- runnable code
- sample input/output
- one short explanation of what concept it reinforces
